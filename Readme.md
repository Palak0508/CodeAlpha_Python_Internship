# CodeAlpha Python Programming Internship

This repository contains the projects completed during my Python Programming Internship at CodeAlpha. Each project focuses on core programming principles, logic building, and professional file management.

---

## Project 1: Hangman Game
A text-based implementation of the classic Hangman game where the user guesses a hidden word letter by letter.
* **Key Concepts**: Random selection, while loops, conditional logic, user input validation, and list operations.
* **File**: `hangman.py`

---

## Project 2: Stock Portfolio Tracker
A command-line tool that simulates a stock portfolio, allowing users to input stocks and quantities to calculate total investment values based on a predefined price system.
* **Key Concepts**: Dictionary structures, numeric input casting, basic arithmetic operations, error handling, and file creation.
* **File**: `stock.py`
* **Output**: Generates a summary text file directly to the user's Desktop environment.

---

## Project 3: Task Automation Script
A practical script designed to organize files within a local directory automatically. It scans a source directory and moves specified files to a targeted location.
* **Key Concepts**: File system navigation, directory generation, string filtering, and automated file transfers using standard libraries.
* **File**: `automation.py`

---

## Project 4: Basic Chatbot
A rule-based chatbot capable of parsing user queries and providing text responses based on specific key phrases.
* **Key Concepts**: Text normalization, whitespace trimming, string membership checking, and continuous interaction loops.
* **File**: `chatbot.py`

---

To run any of these scripts locally, open a terminal environment inside the project folder and execute the file using Python:
